Storage Solutions

Comprehensive storage portfolio supporting distributed, unified, and high-performance scenarios. From Ceph-based distributed storage to NVMe-optimized performance tiers.

ZStone

Distributed Block Storage

Ceph-based distributed storage providing scalable, resilient block storage for virtual machines and applications.

  • Block storage only
  • Self-healing clusters
  • Automatic replication
Best for: HCI, Cloud environments

ZCE

Unified Storage Platform

Multi-protocol storage supporting block, file, and object workloads in a single unified platform.

  • Block + File + Object
  • NFS / SMB support
  • S3-compatible object
Best for: Mixed workloads, Enterprise NAS

ZBS

High-Performance Storage

NVMe-optimized storage architecture designed for databases and latency-sensitive applications.

  • NVMe-oF support
  • Sub-millisecond latency
  • CBD optimized
Best for: Databases, High-performance workloads

ZStack ZAKU

MetaVirt Kubernetes Unit

Enterprise Kubernetes platform fully integrated with ZStack Cloud. Deploy, manage, and scale containerized applications with the same operational model as your virtual machines.

Platform Features

  • Integrated with ZStack Cloud networking (VPC, security groups)
  • Persistent volume integration with ZStack storage
  • Horizontal pod autoscaling (HPA)
  • Cluster autoscaling for node provisioning
  • Built-in monitoring and logging stack
  • Helm chart marketplace integration
